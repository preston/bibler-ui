Developer API

Bibler supports a RESTful JSON API for developing your own mashups.

Developers are free to integrate Bibler into custom applications. It's fine to use the public site for light development and testing purposes, but you'll be blocked if you make excessive requests. If you're integrating with a heavy production app, you need to deploy your own copy of Bibler.

GET the global bible list

{{biblerService.getUrl()}}/bibles.json 

					{{stringify(bibles)}}

GET the global book list

{{biblerService.getUrl()}}/books.json 

					{{stringify(books)}}

GET chapter list for a bible book

{{biblerService.getUrl()}}/{{bible.slug}}/{{book.slug}}.json 

					{{stringify(chapters)}}

GET all verses of a bible book

{{biblerService.getUrl()}}/{{bible.slug}}/{{book.slug}}/{{chapter}}.json 

					{{stringify(verses)}}

POST a full text bible search

{{biblerService.getUrl()}}/{{bible.slug}}/search.json

parameters
text: 'jesus christ' // required
